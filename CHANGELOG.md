Initial public draft release.

- Core ACP README
- Formal protocol specification

- Context request example
- Signed context packet example
- Conformance model
- Security policy
- Roadmap
- RFC series:

  - RFC 0004: Trust Model

ACP Draft v0.1 is a request-for-comments draft. It is not yet a final standard.
