# Code of Conduct

## Our Pledge

We are committed to a respectful, open, and constructive environment for everyone participating in Agent Context Protocol (ACP).

## Expected Behavior

Participants are expected to:

- be respectful and professional
- focus on protocol quality and user safety
- welcome implementation feedback
- disclose conflicts of interest when relevant
- avoid harassment, abuse, personal attacks, or exclusionary behavior

## Unacceptable Behavior

Unacceptable behavior includes:

- harassment or intimidation
- discriminatory language
- threats or abuse
- publishing private information without consent
- intentionally disrupting protocol review

## Enforcement

Maintainers may remove, edit, or reject comments, commits, issues, and pull requests that violate this code of conduct.

## Scope

This code applies to repository discussions, issues, pull requests, documentation, community review, and related ACP project spaces.
