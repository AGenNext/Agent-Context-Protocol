# Contributing

Thank you for contributing to Agent Context Protocol (ACP).

## Goals

ACP aims to define an open, vendor-neutral protocol for governed context delivery.

Contributions should improve:

- clarity
- interoperability
- security
- auditability
- conformance
- federation

## Contribution Types

- specification improvements
- examples
- conformance tests
- security reviews
- implementation reports
- interoperability reports

## Pull Requests

Every pull request should include:

- problem statement
- proposed change
- compatibility impact
- security impact
- conformance impact

## Protocol Changes

Normative protocol changes should be proposed through RFC documents.

## Principles

- vendor neutral
- implementation neutral
- security first
- least privilege
- auditability by default
- explicit trust boundaries
